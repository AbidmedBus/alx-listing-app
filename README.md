# ALX Listing App

مشروع واجهة عقارات على شكل Airbnb باستعمال Next.js، TypeScript، TailwindCSS.

## 📁 هيكلة المشروع

- `components/`: مكونات الواجهة (Card, Button).
- `interfaces/`: أنواع البيانات TypeScript.
- `constants/`: ثوابت عامة مثل العناوين أو روابط.
- `public/assets/`: صور ومحتويات ثابتة.

## ▶️ كيفاش تشغل المشروع

1. ثبت الحزم:
```bash
npm install
```

2. شغل السيرفر:
```bash
npm run dev
```

3. فتح فـ المتصفح: [http://localhost:3000](http://localhost:3000)
